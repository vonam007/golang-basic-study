# Topic 04 – Operators

## Arithmetic Operators

| Operator | Mô tả  | Ví dụ                                |
| -------- | ------ | ------------------------------------ |
| `+`      | Cộng   | `5 + 3 → 8`                          |
| `-`      | Trừ    | `5 - 3 → 2`                          |
| `*`      | Nhân   | `5 * 3 → 15`                         |
| `/`      | Chia   | `7 / 2 → 3` (int), `7.0 / 2.0 → 3.5` |
| `%`      | Modulo | `7 % 3 → 1`                          |

## Comparison Operators

`==`, `!=`, `<`, `>`, `<=`, `>=` — trả về `bool`.

## Logical Operators

`&&` (AND), `||` (OR), `!` (NOT)

## Bitwise Operators

| Operator | Mô tả       |
| -------- | ----------- |
| `&`      | AND         |
| `\|`     | OR          |
| `^`      | XOR         |
| `<<`     | Left shift  |
| `>>`     | Right shift |
